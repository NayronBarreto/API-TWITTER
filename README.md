# API-TWITTER
This API show the best practice to create a REST API.

Using:
 express
 jsonwebtoken
 mongoose"
 
Install:
    npm install.

Routes:

post('/signup');
post('/signin');


post('/tweets');
delete('/tweets/:id');

post('/like/:id');

get('/users/me');
get('/feeds');

post('/follow/:id');
delete('/unfollow/:id');
