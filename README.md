# API-TWITTER
This API show the best practice to create a REST API.

Using:
 express
 jsonwebtoken
 mongoose"
 
Install:
    npm install.

Routes:

post('/api/signup');
post('/api/signin');


post('/api/tweets');
delete('/api/tweets/:id');

post('/api/like/:id');

get('/api/users/me');
get('/api/feeds');

post('/api/follow/:id');
delete('/api/unfollow/:id');
